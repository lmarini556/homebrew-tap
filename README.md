# Homebrew Tap

Personal Homebrew tap.

## Available Casks

### AIM — Agent Instances Manager

```bash
brew install lmarini556/tap/aim
```

Native macOS app for managing Claude Code instances. Apple Silicon only.

**Source**: [lmarini556/aim](https://github.com/lmarini556/aim)

## Adding This Tap

```bash
brew tap lmarini556/tap
```

Then install any cask by name.
